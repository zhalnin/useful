Тестовая публикация

08 мая 2012 года
Вышла новая версия Redsn0w 0.9.10b8b, которая позволяет сделать привязанный джейлбрейк на iOs 5.1.1, 5.1, 5.0, 4.3, 4.2.1, 4.3.5 и не привязанный на прошивки 4.1, 4.3.3, 5.0.1 . Последние обновления redsn0w: Update #2: Версия 0.9.10b7 - выполняет непривязанный джейлбрейк iPhone4S and iPad2 с прошивкой 5.0.1 И ТОЛЬКО НА ЭТОЙ ПРОШИВКЕ. - с его помощью можно сделать повторный джейлбрейк. - при подписывании IPSW, позволяет использовать подпись прямо из Cydia - показывает больше информации о вашем аппарате (например: iPhone3G имеет уязвимый загрузчик, или же iPhone3GS имеет старый загрузчик Update #3: Версия redsn0w 0.9.10b8 - добавляет мозможность сделать backup произвольных директорий или фалов на вашем аппарате путем создания сжатого zip-архива на PC или Mac - добавлена новая кнопка Extras->Even More->Backup, необходимо, чтобы на аппарате был джейлбрейк и активирована служба afc2(почти у всех аппаратов с джейлбрейком она уже активирована) - по-умолчанию делается бекап активационной записи по пути: /var/root/Library/Lockdown, которая полезна для тех, кто использует активацию посредством SAM (больше информации об этом обещано в следующих обновлениях). Update #3b: Версия redsn0w 0.9.10b8b создает файлы с расширением .zip, которорые более удобны для Windows explorer (известно, что Windows не привечает слеш с наколоном вправо (*.nix) в названии файлов. Для 3GS cо старым бутромом - джейлбрейк НЕ ПРИВЯЗАННЫЙ. Привязанный джейлбрейк iOS 5.1.1: 1. Запустить redsn0w 2. Идем Extras -- Select IPSW и выбираем iOS 5.1 3. Далее идем обратно и нажимаем Jailbreak 4. Ставим галку на Cydia 5. После автоматической перезагрузки аппарат останется в DFU 6.
